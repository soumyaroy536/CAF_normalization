# Company Name Normalization

This project normalizes inconsistent company names from Excel data.

## Problem
Company names appear in different formats such as:
- CAF softsol
- CAF solution
- CAF softSolution India Pvt Limited

## Solution
A Python function cleans and standardizes the company name.

## Output
CAF SoftSol India Pvt Ltd.
